# Vagrant

## Prerequisites 

```
sudo zypper install vagrant
sudo usermod -G libvirt -a $USER
```
## Usage of Vagrant box
```
vagrant up

```
* Vagrant supports HTTP_PROXY and HTTPS_PROXY environmental variables.
